# Vision Guide

How to install ?

# 1. 

```bash
git clone git@github.com:sachamutschler/vision-guide.git
```
(ssh key needed)

# 2. 

copy paste .env file from discord (channel note et ressources) in the root project.

# 3.

run the following command : 

```bash
docker compose up
```

# 4. 

all good, you can start to work !
